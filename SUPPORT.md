# Support

Agent Board is a small open-source local tool. Support is issue-based and best effort.

## Ask A Question

Use GitHub issues for questions about setup, adapting the board, or unclear documentation:

https://github.com/Th3Sp00kyM8/agent-board/issues

Include:

- Your operating system
- Node.js version from `node --version`
- The command you ran
- The exact error text or screenshot

## Report A Bug

Open a bug issue when something breaks in a fresh clone or template-created repo.

Before reporting, run:

```powershell
npm run verify
npm run build
```

Attach the failing output if either command fails.

## Security Issues

Do not open a public issue for a security-sensitive report. Follow `SECURITY.md` instead.

## Out Of Scope

This project does not provide hosted service support, account management, cloud sync, or private customization work through the public issue tracker.
