# Code of Conduct

Agent Board uses a simple contributor standard: be respectful, stay on topic, and assume good intent while keeping feedback specific and actionable.

## Expected Behavior

- Use welcoming and inclusive language
- Respect different experience levels and project contexts
- Keep issue and pull request discussion focused on the tool
- Give technical feedback with enough detail for someone to act on it
- Accept maintainer decisions when a request is out of scope

## Unacceptable Behavior

- Harassment, threats, insults, or personal attacks
- Repeated off-topic comments after being asked to stop
- Sharing another person's private information
- Spam, low-effort promotion, or unrelated support requests

## Reporting

If behavior in the project becomes a problem, open a private security advisory only if private information or abuse-sensitive details are involved. Otherwise, open an issue with a clear summary and links to the relevant discussion.

Maintainers may edit, hide, lock, or remove comments and issues that violate this standard.
